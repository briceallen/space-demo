PROJECT OVERVIEW
────────────────
"Endless Horizons" is a civilization builder game inspired by Endless Space,
built entirely with Three.js (v0.164.0), Tailwind CSS, and vanilla ES modules.
No build pipeline — served via npx serve or python -m http.server.

  - clock = new THREE.Clock()

  Galaxy branch (viewLevel === "galaxy"):
    1. WASD flying: Move galaxyCam.center along camera-relative axes
       Speed = dt * radius * 0.5 (scales with zoom)
    2. Orbit camera: position from getGalaxyCameraPosition()
    3. Current ring: lookAt camera, pulse opacity sin(t*3)
    4. Hover ring: lookAt camera, pulse opacity sin(t*5), scale pulse sin(t*4)
    5. Trail line: pulse opacity sin(t*2)
    6. Star breathing:
       Glow: 0.18 + 0.1*sin(t*1.5 + phase)
       Bloom: 0.04 + 0.03*sin(t*0.8 + phase*2)
       Flare: (0.25 + 0.15*sin(t*2.0 + phase*3)) * sizeMultiplier
    7. Hover raycasting (when not dragging):
       Raycast mouseNDC against all core meshes
       Highlight: scale 1.4x, emissive 3.0, show ring + tooltip
       Unhighlight: restore scale + emissive 1.5
    8. Tooltip positioning: follows mouse + 16px right, -10px up

  System branch:

    2. Auto-rotation (AUTO_ROTATE_SPEED * (1 + i*0.25))
    3. Drag rotation (quaternion-based)
    4. Return animation (slerp back to neutral)
    5. Axial tilt application
    6. Background star twinkle (sine-based alpha + size)
    7. Camera position/look lerp (LERP_SPEED)
    8. Game tick (if playing, every 1.0 seconds)
